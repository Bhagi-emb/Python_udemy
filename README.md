Precedence	        Operator	    Description
==================================================================================
1 (Highest)	          ()	        Parentheses (grouping)
2                     **	        Exponentiation

4	              *, /, //, %	    Multiplication, Division, Floor Division, Modulo
5	                  +, -	        Addition, Subtraction
6 (Lowest)	    Comparisons, assignment, etc.	Used after arithmetic
